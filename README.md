# Power Platform Portfolio

A responsive GitHub Pages portfolio for Power Apps, Power BI, and workflow automation.

## Files

- `index.html` — page structure and content
- `styles.css` — visual identity and responsive design
- `script.js` — mobile navigation, project tabs, and reveal animations

## Publish on GitHub Pages

1. Upload all three website files to the root of your `username.github.io` repository.
2. Open repository **Settings** → **Pages**.
3. Select **Deploy from a branch**.
4. Choose `main` and `/root`.
5. Save.
